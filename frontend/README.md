# Módulos del Frontend 
  La aplicación web se divide en dos módulos principales: 
  • Módulo público informativo 
  • Módulo administrativo 
 Se contempla futuramente un módulo de autenticación (inicio de sesión). 

# Módulo público (usuario general) 
Pantalla principal 

Contenido: 
• Nombre del gimnasio 
• Banner informativo 
• Descripción general 
• Menú de navegación con accesos a: 
  o Máquinas 
  o Servicios 
  o Productos 
  o Ubicación 
• Enlaces a redes sociales 

# Pantalla de máquinas 

Contenido: 
• Galería de imágenes 
• Información de cada máquina: 
  o Nombre 
  o Área a la que pertenece 
• Botón para regresar a inicio 

# Pantalla de servicios 

Contenido: 
• Opciones para visualizar: 
o Entrenamientos personalizados 
o Planes y membresías 
  • Descripción y precios 
• Botón para regresar a inicio 

# Pantalla de productos 

Contenido: 
• Categorías: 
    o Suplementos 
    o Ropa deportiva 
    • Descripción y precio de cada producto 
    • Botón para regresar 
    
# Pantalla de ubicación 

Contenido: 
• Mapa 
• Información del gimnasio: 
o Dirección 
o Horarios 

# Módulo administrativo 
Panel principal 
Accesos a gestión de: 
• Máquinas 
• Servicios: 
  o Entrenamientos personalizados 
  o Membresías 
• Productos: 
  o Suplementos 
  o Ropa deportiva 
  
# El administrador podrá: 
• Agregar 
• Modificar 
• Eliminar información
